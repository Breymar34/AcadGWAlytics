About This App

AcadGWAlytics : Academic Data Mining Analytics for GWA Prediction

This web application helps predict the General Weighted Average (GWA) of students based on their academic records, entrance exam scores, and background information. It uses a linear regression model to generate predictions and provides performance metrics such as R², RMSE, and MAE.
